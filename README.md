# Iron ERP Dashboard — Ferrum Logic

A professional steel supply and ERP management web application built for **Ferrum Logic**.

## 🚀 Features

- **React 18** — Modern rendering with concurrent features
- **Vite** — Lightning-fast build tool and development server
- **Redux Toolkit** — Simplified state management
- **TailwindCSS** — Utility-first CSS framework
- **React Router v6** — Declarative client-side routing
- **Framer Motion** — Smooth UI animations
- **React Three Fiber** — 3D product visualizations
- **React Hook Form** — Efficient form handling

## 📋 Prerequisites

- Node.js (v16.x or higher)
- npm

## 🛠️ Installation

```bash
npm install
npm run dev
```

## 📁 Project Structure

```
iron-erp-dashboard/
├── public/             # Static assets
├── src/
│   ├── components/     # Reusable UI components
│   ├── pages/          # Page components
│   ├── styles/         # Global styles
│   ├── App.jsx         # Main application component
│   ├── Routes.jsx      # Application routes
│   └── index.jsx       # Application entry point
├── index.html          # HTML template
├── package.json        # Dependencies and scripts
├── tailwind.config.js  # Tailwind CSS configuration
└── vite.config.js      # Vite configuration
```

## 🧩 Available Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start local development server |
| `npm run build` | Build for production (GitHub Pages) |
| `npm run serve` | Preview production build locally |

Built with a mobile-first approach using Tailwind CSS breakpoints for all screen sizes.
